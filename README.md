# 技能斗地主（3人，2真人+机器人）

这是一个可联机的浏览器小游戏：**2 个真人 + 1 个机器人**，通过分享链接（可发微信）进入同一房间。

## 功能（当前版本）

- 房间：创建/加入/复制邀请链接
- 3 人局：2 真人 + 1 机器人
- 斗地主完整规则：
  - 叫分/抢地主、发牌+底牌归属
  - 完整牌型：单/对/三/三带一/三带二/顺子/连对/飞机(带翅膀)/四带二/炸弹/王炸
  - 出牌比较与轮次结算（过牌、管牌、回合重置）
  - 加倍：地主加倍、农民加倍
  - 春天/反春
  - 计分与结算
  - 超时托管、自动过/自动出最小牌
  - 断线重连（基于 playerToken）
- 技能（每人开局选 1 个，每局限用 1 次）：
  - **点数+1**：在自己回合出牌前，选择手里任意一张牌（3~A），点数 +1（A→2；2/王不可加）
  - **底牌抽2**：在底牌揭示阶段从 3 张底牌里抽 2 张加入手牌，剩余 1 张归地主

## 部署

- 前端（静态）：GitHub Pages → `web/` 目录
- 后端（Node.js）：Render → `server/` 目录

## 联机方式

- 部署后访问：`https://shuning302.github.io/doudizhu?backend=https://your-render-url.onrender.com`
- 你点击"创建房间"，复制邀请链接发给朋友
- 朋友用手机/电脑浏览器打开链接即可加入
