# 技能斗地主（3人，2真人+机器人）

这是一个可联机的浏览器小游戏原型：**2 个真人 + 1 个机器人**，通过分享链接（可发微信）进入同一房间。

## 功能（当前版本）

- 房间：创建/加入/复制邀请链接
- 3 人局：2 真人 + 1 机器人
- 斗地主最小闭环（简化版出牌类型）：
  - 牌型：单张、对子、三张、炸弹、王炸
  - 暂不支持：顺子、连对、飞机、三带一/二 等
- 技能（每人开局选 1 个，每局限用 1 次）：
  - **点数+1**：选择你手里任意一张牌（3~A），点数 +1（A→2；2/王不可加）
  - **底牌抽2**：在“发底牌阶段”从 3 张底牌里随机抽 2 张加入自己手牌，剩余 1 张留给地主

## 运行

需要本机已安装 Node.js（建议 18+）。

```bash
cd doudizhu-skills
npm install -w server
npm run dev
```

打开 `http://localhost:5179`。

## 联机方式（发微信邀请）

- 你点击“创建房间”
- 点“复制邀请链接”，把链接发给朋友（微信）
- 朋友用手机/电脑浏览器打开链接即可加入

> 如果你们不在同一局域网：需要把服务部署到公网（后续我也可以帮你加一键部署方案）。
